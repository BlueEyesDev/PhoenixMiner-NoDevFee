- **PhoenixMiner 5.***

- **ETH**

### Compatible Pools
/!\ Only Stratum Port /!\
- **ethermine.org**
- **ethpool.org**
- **nanopool.org**
- **2miners.com**
- **sparkpool.com**

### How to use :

### Linux :
Install redirect
```
sudo node Install.js
```
Use
```
sudo node Main.js eu1.ethermine.org 0x1279b6dD477b1F0daBD7c7aB1Cd3b6Ccc1546D7F
```
To Uninstall the redirect
```
sudo node Uninstall.js
```
To AutoUpdate Pools
```
sudo node Uninstall.js
sudo node UpdatePools.js
sudo node Install.js
```
